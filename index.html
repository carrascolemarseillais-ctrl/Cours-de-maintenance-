III-Les caractéristiques de Linux

  1. DEFINITION DE LINUX 🐧

    2. LINUX est un système d’exploitation libre et open source basé sur le noyau Linux, qui
    permet de faire fonctionner un ordinateur, gérer le matériel (processeur, mémoire, disque, etc.) et exécuter
    des programmes.

  2. présentation de sa création

      • l'histoire de LINUX

        • Linux a été créé en 1991 par LINUS TORVALDS, un étudiant finlandais en informatique.
        Au départ, il s’agissait d’un projet personnel, mais il est devenu aujourd’hui l’un des systèmes
        d’exploitation les plus utilisés dans le monde.

      • les différentes versions de Linux ( les prinipales distribution)
        1. Ubuntu
          • Une des versions les plus populaires.
          • Facile à utiliser pour les débutants.
          • Très utilisée pour les ordinateurs personnels et les serveurs.

        2. Debian
          • Distribution très stable et fiable.
          • Beaucoup utilisée pour les serveurs.
          • Ubuntu est basé sur Debian.

        3. Fedora
          • Version moderne avec les nouvelles technologies Linux
          • Soutenue par l’entreprise Red Hat.

        4. Linux Mint
          • Très simple pour les débutants venant de Windows.
          • Interface proche de Windows.

        5. Kali Linux
          • Utilisé pour la cybersécurité et le hacking éthique
          • Contient beaucoup d’outils de sécurité.

        6. Arch Linux
          • Distribution très personnalisable
          • Plutôt utilisée par les utilisateurs avancés.

        7. Red Hat Enterprise Linux
          • Version professionnelle pour les entreprises et serveurs.
          • Payante avec support technique.

NB: Un point clé a retenir sur les système d'exploitations
Il faut retenir que les systèmes d'exploitation (OS) sont des logiciels qui
gérent les recourrces d'un ordinateur et fournisentdes sercives aux programmes.
voici quelques points clés:

  • Gérent les resources matérielles (CPU,mémoire,disquedure,etc.)
  • Fournisent une interface utilisateur (CLI ou GUI)
  • Gérent les fichiers et les répertoires
  • Contrôlent les éntrées/sorties (clavier,souris,écran,etc.)
  • Gérent le processus et les tâches
  • Assurent la sécurité et la gestion des droits d'acèes
